# Торговая платформа «Зайди удивишься» (TM Trade Platform)

Демонстрационный прототип цифровой торговой платформы для Туркменистана.

## Функциональность

- Авторизация с трёхфакторной аутентификацией (телефон, email, электронный ключ)
- Товарная биржа «Алыш-Чалыш» с 4 категориями (по 6 товаров в каждой)
- Корзина с розничными и оптовыми ценами
- Встроенный мессенджер и почтовый сервер
- Платёжная система и криптовалюта TM Coin (симуляция)
- Админ-панель с редактированием товаров
- Поддержка трёх языков: русский, туркменский, английский

## Технологии

- Чистый HTML, CSS (Tailwind), JavaScript
- Локальное хранение данных (localStorage)
- Адаптивная вёрстка

## Как запустить локально

1. Скачайте файл `index.html`
2. Откройте его в браузере
3. Войдите как администратор:  
   - **Логин:** `admin@dtm.tm`  
   - **Пароль:** `12345678`
